# Проект для сдачи домашнего задания по теме "Работа с API"

## Описание проекта
Учебный проект для освоения принципов взаимодействия с REST API. Реализация включает выполнение HTTP-запросов, обработку JSON-ответов и создание пользовательского интерфейса для работы с различными API.

## Реализованные требования

### ✅ Обязательные задания (минимум):
1. **Универсальный GET-запрос** - возможность обращаться к любому API по указанному URL
2. **Парсинг JSON-ответов** - извлечение и форматирование данных из API-ответов
3. **Работа со странами** - интеграция с RestCountries API для получения информации о странах
4. **Выделение ключевых полей** - демонстрация извлечения конкретных данных (например, температура из погодного API)

### ✅ Дополнительные задания (средний уровень):
1. **Модуль для случайных собак** - интеграция с Dog CEO API (`https://dog.ceo/api/breeds/image/random`)
2. **Выделенный HTTP-клиент** - модуль `http_client.py` с функциями `get()`, обработкой статусов и таймаутами
3. **Локализованное меню** - русскоязычный интерфейс с пунктами:
   - 1 - GET запрос по URL
   - 2 - Информация о стране  
   - 3 - Случайная собака
   - 0 - Выход
4. **Красивый вывод** - использование colorama для цветного форматирования в консоли

## Техническая реализация

### Структура проекта:
```
api_homework/
├── main.py                 # Основной файл с меню и логикой
├── http_client.py          # Модуль для HTTP-запросов (GET)
├── country_info.py         # Форматирование информации о странах
├── json_pretty_printer.py  # Универсальный красивизатор JSON для любых API ответов
├── weather_example.py      # Пример выделения конкретных полей из JSON ответа
├── requirements.txt        # Зависимости: requests, colorama
└── README.md               # Документация
```

### Ключевые функции:
1. **GET-запросы к любым API** - через функцию `make_get_request()`
2. **Специализированные запросы**:
   - `get_country_info()` - данные о странах
   - `get_random_dog()` - случайные изображения собак
3. **Обработка ошибок** - таймауты, HTTP-статусы, JSON-парсинг
4. **Пользовательский интерфейс** - интерактивное консольное меню

### Используемые API (демонстрационные примеры):

1. **Dog CEO API** (`https://dog.ceo/api/breeds/image/random`) - получение случайных изображений собак с информацией о породах
2. **RestCountries API** (`https://restcountries.com/v3.1/name/{country}`) - географические, демографические и культурные данные о странах мира
3. **FakeStore API** (`https://fakestoreapi.com/products/{id}`) - тестовые данные товаров для e-commerce приложений
4. **Chuck Norris Jokes API** (`https://api.chucknorris.io/jokes/random`) - генератор случайных шуток о Чаке Норрисе
5. **The Cat API** (`https://api.thecatapi.com/v1/images/search`) - коллекция изображений кошек

**Важно**: Проект поддерживает работу с любым JSON API - достаточно указать URL эндпоинта в соответствующем пункте меню ("Своя ссылка").

## Инструкция по запуску

### 1. Установка зависимостей:
```bash
pip install -r requirements.txt
```

### 2. Запуск программы:
```bash
python main.py
```

### 3. Примеры использования:
```
Выберите пункт меню:
1. GET запрос по URL        # Любой API
2. Информация о стране      # Например: Italy, Germany, France
3. Случайная собака         # Ссылка на изображение
0. Выход
```
